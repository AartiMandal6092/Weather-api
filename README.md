# 🌦️ Weather CLI (Python)

A simple command-line Python script that fetches current weather data for a given city using the Open-Meteo API.

---

## 🚀 What it does

* Takes a city name as input
* Shows multiple matching locations (if available)
* Fetches real-time weather data
* Displays:

  * Temperature
  * Wind speed
  * Wind direction

---

## 🛠️ Tech Used

* Python
* requests
* Open-Meteo API

---

## ▶️ How to Run

```bash
pip install requests
python weather.py
```

---

## 📌 Example

```text
Enter the location: Delhi

Select the correct location:

0 Delhi - Delhi - India
1 Delhi - California - USA

Enter number: 0

Weather Details of Delhi:

Current Temperature: 32 °C
Wind Speed: 12 km/h
Wind Direction: 240
```
